Sizing Units

  • Absolute Units

    These units are basically fixed sizing units, they don't rely on body or viewport

    • px - Pixel
    • cm - Centimeter
    • in - Inches
    • mm - Millimeters

  • Relative Units

    These units are basically dependent on other factors like parent element,viewport etc

    • % - Percentage
    • vw - Viewport width
    • vh - Viewport height
    • em - Relative to parent
    • rem - Relative to root i.e html
